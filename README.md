🐧 Linux Basic Commands

This file contains basic Linux commands that I am learning while building my foundation in networking and cybersecurity.

---

📘 Command List

Command| Use| Example
"pwd"| Shows current directory path| "pwd"
"ls"| Lists files and directories| "ls"
"ls -l"| Shows detailed file information| "ls -l"
"ls -a"| Shows hidden files| "ls -a"
"cd"| Changes directory| "cd /home"
"cd .."| Moves one directory back| "cd .."
"cd ~"| Goes to home directory| "cd ~"
"mkdir"| Creates a new directory| "mkdir test"
"rmdir"| Removes empty directory| "rmdir test"
"touch"| Creates an empty file| "touch file.txt"
"cp"| Copies files| "cp file.txt backup.txt"
"cp -r"| Copies directories recursively| "cp -r folder1 folder2"
"mv"| Moves or renames files| "mv file.txt new.txt"
"rm"| Deletes a file| "rm file.txt"
"rm -r"| Deletes a directory recursively| "rm -r folder"
"cat"| Displays file content| "cat file.txt"
"less"| Views file page by page| "less file.txt"
"head"| Shows first 10 lines of a file| "head file.txt"
"tail"| Shows last 10 lines of a file| "tail file.txt"
"grep"| Searches text inside files| "grep error file.txt"
"wc"| Counts lines, words, characters| "wc file.txt"
"sort"| Sorts file content| "sort file.txt"
"chmod"| Changes file permissions| "chmod 755 script.sh"
"chown"| Changes file owner| "chown user file.txt"
"whoami"| Shows current logged-in user| "whoami"
"uname -a"| Shows system information| "uname -a"
"clear"| Clears the terminal screen| "clear"
"history"| Shows previously used commands| "history"
"ping"| Tests network connectivity| "ping 8.8.8.8"
"ip a"| Shows IP address information| "ip a"
"top"| Shows running processes| "top"
"ps aux"| Shows detailed process list| "ps aux"
"kill"| Stops a running process| "kill 1234"
